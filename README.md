# قالب بورتفوليو بسيط (نسخة مفتوحة المصدر)

هذا قالب بسيط ومرتب للمبرمجين وعلماء البيانات والمهندسين. مبني بـ HTML و CSS و JavaScript الأساسية، يعني خفيف وما يحتاج أي تعقيد.

## ليش تستخدمه؟

- تصميم نظيف يركز على المحتوى.
- فيه لوحة تحكم تخليك تضيف مشاريعك وشهاداتك بكل سهولة.
- تقدر تستورد مشاريعك من قيت هب (GitHub) بضغطة زر.
- يشتغل بامتياز على الجوال والكمبيوتر.
- ما يحتاج سيرفر ولا قاعدة بيانات، تقدر ترفعه كـ "Static Site" في أي مكان.
- سهل في التخصيص، بس تعدل ملف واحد وتحط صورك.

---

## كيف تبدأ؟ (3 خطوات بس)

### 1. زبّط مجلد بياناتك

كل بياناتك تكون في مجلد my-data:
- ملف config.js : إعداداتك (انسخها من config.example.js).
- ملف portfolio_data.json : بيانات مشاريعك وشهاداتك.
- مجلد images : حط صورك فيه.

### 2. عدّل الإعدادات

خذ نسخة من config.example.js وسمّها my-data/config.js وحط فيها بياناتك. في هذي النسخة، تقدر تدخل لوحة التحكم بدون كلمة مرور عشان تجربها براحتك.

### 3. ارفع موقعك

ارفعه على GitHub Pages أو أي استضافة ثابتة، والموضوع خالص.

---

## كيف تتحكم في المحتوى؟

### عن طريق لوحة التحكم الجانبية (أسهل لك)
1. افتح الموقع، وانزل تحت واضغط "إدارة الموقع".
2. بتطلع لك لوحة جانبية تقدر تضيف فيها كل شي (سيرتك، مشاريعك، شهاداتك).
3. لما تخلص، اضغط "Export JSON" عشان يحمل لك ملف البيانات الجديد، خذه وحطه في مجلد my-data عشان تثبت التغييرات.

### التعديل اليدوي
إذا تحب الأكشن، تقدر تعدل ملف my-data/portfolio_data.json مباشرة بأي محرر نصوص.

## التقنيات المستخدمة
- HTML5 و CSS3 و JavaScript (الشي الأساسي بدون فريمورك).
- البيانات تنحفظ وتستخرج بصيغة JSON.
